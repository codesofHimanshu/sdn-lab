# Assignment 1: TCP Client-Server

## 📌 Objective
To implement basic TCP communication between a client and server using sockets in C++.

## ⚙️ Implementation Details
- Server listens on a specific port (e.g., 5000)
- Client connects to server using IP address
- Client sends message to server
- Server receives and responds
